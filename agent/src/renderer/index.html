NetPilot Agent Help

Getting Started

  1. Enter your router's IP address (usually 192.168.1.1)
  2. Enter the username (usually "root" for OpenWrt)
  3. Enter the router password
  4. Click "Test Connection" to verify access
  5. Click "Install & Connect" to set up NetPilot

Troubleshooting

Connection Failed
Check that the router IP is correct and the router is powered on. Ensure SSH is enabled on the router.
Package Installation Failed
Make sure the router has internet access and sufficient storage space. Try updating package lists first.
Tunnel Establishment Failed
Verify cloud VM is accessible and firewall allows the tunnel port. Check network connectivity.

Features

  • Test Connection: Verify router accessibility
  • Enable WiFi: Enable default wireless interface
  • Install & Connect: Full NetPilot setup
  • Reconnect: Re-establish tunnel without reinstalling
  • Uninstall: Remove NetPilot components from router
